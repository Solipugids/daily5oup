CentOS 7中将普通用户列入root组

一般初始情况下，CentOS 7的普通用户如果想要执行sudo相关命令的话，每次都要用su命令切换到root用户下执行。常规新增的普通用户没有在root组下，所以需要通过修改/etc/sudoers文件将指定用户加入root组。修改该文件可以使用visudo命令。

  1. 切换到root

    用su -命令切换到root组。和su命令不同的是，su -命令可以将环境变量一起传递过来。

  2. 修改sudoers文件
