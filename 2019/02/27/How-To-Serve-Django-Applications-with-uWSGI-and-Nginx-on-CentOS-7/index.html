How To Serve Django Applications with uWSGI and Nginx on CentOS 7

Posted on 2019-02-27 | Views:

Django在本地测试环境运行比较简单，部署好之后直接运行以下命令即可：

python manage.py runserver 0.0.0.0:8000

但是在生产环境下不会这么做。
Django在官方网站上，有关于如何利用WSGI部署生产项目的详细说明。在centOS 7服务器上发现，WSGI部署本身比较容易实现，但是WSGI和Nginx联用就容易出错。在实践过程中按照很多网上博客的内容执行没有成功过，这可能和软件版本、系统环境等因素有关系（具体出错的原因无法追溯）。为了容易重复以下的部署过程，列出相关版本参数：

Python 2.7.5
Django 1.11.13

注：查看Django版本的方法：

进入python命令行
import django
django.get_version()
# django # uwsgi # centos # nginx
